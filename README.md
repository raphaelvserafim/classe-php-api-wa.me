### EXAMPLES


#####  WHATSAPP
```php

use Api\Wame\WhatsApp;

include_once 'vendor/autoload.php';

$whatsapp     = new WhatsApp(
    ["server" => "API server", 
    "key" => "Your Key Instance"]
    );
```

#### Get QrCode HTML
```php

```

#### Get QrCode Base64
```php

```

#### Infor Instance

